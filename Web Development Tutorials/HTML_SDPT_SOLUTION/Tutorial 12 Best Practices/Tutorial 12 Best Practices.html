1. Use proper document structure

- < head > - Metadata, Title, & other information.
- < body > - Content Here

2. Always Specify the !DOCTYPE HTML

- Specifies an HTML 5 Doc

3. Use Lowercase Markup Tags

Example:
< DOCTYPE html />
< html lang="en" />

  < head >
    < meta name="viewport" content"device-width, initial-scale 1.0">
    < title > Title < title >
    < link rel="stylesheet" href= "stylesheet" >
  < head >

  < body >

    < nav >
      < a href= "home.html" /> Home < a />
      < a href= "about.html" /> About < a />
      < a href= "contact.html" /> Contact < a >
    < nav />

    < section >
      < p > Content Here < p />
    < section />
  < body />

  < footer > Insert footer here < footer />
< html />

4. Always close tags

5. Use proper indentation

6. Use only 1 < h /> tag per page

7. Use Semantic Elements where possible

Example:
< DOCTYPE html />
< html lang="en" />

  < head >
    < meta name="viewport" content"device-width, initial-scale 1.0">
    < title > Title < title >
    < link rel="stylesheet" href= "stylesheet" >
  < head >

  < body >

    < nav >
      Content Here
    < nav />

    < header >
      Content Here
    < header />

    < section >
      < p > Content Here < p />
    < section />
  < body />

  < footer >
    < p > © Web Development Journey 2025 < p/>
  < footer />

< html />

9. Use < div > or Semantic Tags to divide the section of your website.
